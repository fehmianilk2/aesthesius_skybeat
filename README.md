**Dionysus** is an intelligent, weather-based music recommendation engine powered by the **Aesthesius** ecosystem. It analyzes real-time weather conditions of any global city and curates a musical mood to match the atmosphere.

## üöÄ Features

* **Real-Time Weather Analysis:** Fetches live data using OpenWeatherMap API.
* **Smart Mood Detection:** Algorithms that map weather conditions (Mist, Rain, Clear) to musical genres.
* **Global City Search:** Built-in support for major world cities + custom search capability.
* **Secure Architecture:** Environment variable protection for API keys.

1.  **Clone the repository:**

    ```bash
    streamlit run app.py
    ```
